Nixie Tube Clock Documentation

【V3】Clock Features

Release time: August 2023

This clock is the third generation Nixie tube clock I made. Compared with the previous two generations, it supports MP3 high-definition voice, brightness adjustment, and Bluetooth control. The new supporting software is more intuitive and easy to use. It has a built-in new generation of sensor switch parameters with adjustable parameters, built-in gyroscope angle recognition, an ultra-thin body of <10mm, and a more advanced 32-bit MCU.

Applicable clock: IN1406Mv3, DGM0106Mv3, IN1606Mv3.0

The clock has the following features:

  • 1: Adopting the former Soviet Union DGM01/IN14 Nixie tube, DGM01 Nixie tube has a longer life and better display effect

  • 2: Bluetooth applet/Android APP is easy to operate, and one-click quick calibration time

  • 3: Built-in new generation induction switch, adjustable sensitivity, more stable performance

  • 4: Built-in gyroscope, can change the backlight and clock switch according to the tilt angle

  • 5: Brightness 10 levels of adjustment, manual automatic adjustment

  • 6: High-quality voice broadcast, you can change the music MP3 by yourself

  • 7: RGB multi-mode color backlight

  • 8: High-precision clock IC, accurate timekeeping

  • 9: Ultra-thin body, solid wood CNC one-piece molded shell

  • 10: Support PC client, can be equipped with PC Bluetooth wireless connection tool

  • 11: Support timer (with countdown) and anniversary date (with countdown days) display

【V2】Clock Features

Release date: October 2020

This clock is the second generation of Nixie tube clock I made. Compared with the previous generation, it supports voice broadcast, brightness adjustment, and for the first time supports Bluetooth control. You can use WeChat applet and Android APP to operate the clock, and the clock setting is more intuitive; it uses a more efficient and stable high-voltage drive circuit and a brand-new drive circuit.

Applicable clocks: IN1406M, IN1206M, IN1404M, IN1204M, IN0806M, IN1806M Bluetooth version (Guoran APP)

The clock has the following features:

  • 1: Adopting former Soviet Union IN14 Nixie tube, long life and good display effect

  • 2: Bluetooth applet/Android APP is easy to operate and can quickly calibrate the time

  • 3: Three-touch button

  • 4: Multi-function 14-button infrared remote control

  • 5: Brightness 10 levels of adjustment, manual automatic adjustment

  • 6: Voice broadcast

  • 7: RGB multi-mode color backlight

  • 8: High-precision clock IC, accurate timekeeping

  • 9: External high-precision temperature sensor

  • 10: Black walnut solid wood integrated CNC molded shell

【V1】Version Clock Usage Instructions

Release date: December 2016 M4 version, August 2017 M6 "sensor switch version"

This clock is the first generation of Nixie tube clocks I made. The in4 version was released in 2016, and the in14|in12 version was released in 2017. The clock uses a CNC integrated solid wood base, and is the first to use 5V efficient and stable high-voltage drive circuit, patented induction switch function, etc. With many innovations, it provides a very good user experience.

Applicable clock: M4 (IN4), M6 (IN14/IN12) induction switch version

The clock has the following features:

  • 1: Adopting former Soviet Union IN14/IN12/IN4 Nixie tube, long life and good display effect

  • 2: The clock has a built-in sensor switch function

  • 3: Three-touch button

  • 4: Three-button infrared remote control

  • 5: Buzzer operation prompt tone

  • 7: RGB multi-mode color backlight

  • 8: High-precision clock IC, accurate timekeeping

  • 9: Black walnut solid wood integrated CNC molded shell

  • 10: High-precision temperature sensor

    (The above is a brief introduction to the IN14/12 clock function)
